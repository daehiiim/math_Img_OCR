Handoff Summary — mathOCR Project
Project Overview
A math OCR web application with:
- Backend: FastAPI (Python) at D:/project/mathOCR/math_Img_OCR-main/app/ — port 8000
- Frontend: React + Vite + TypeScript at D:/project/mathOCR/mathOCR_design/ — port 5173
- HWPX Generator: D:/project/mathOCR/hwpx_generator/ — converts OCR results to Korean word processor format
- Sample Data: D:/project/mathOCR/DAE_sampleData/imgExample.hwpx
Pipeline Flow
1. User uploads math problem image → backend creates job
2. User draws regions on image (RegionEditor) → saves to backend
3. Backend runs OCR pipeline per region → produces OCR text, SVG, MathML, explanation
4. User can edit SVG shapes (SvgCanvasEditor) → saves edited SVG
5. User exports to HWPX → backend renders SVG→PNG, generates HWPX document
