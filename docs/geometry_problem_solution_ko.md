# 도형 문제 OCR + 정리 + 풀이

## 1) 문제 OCR (이미지 텍스트 추출)

3. △ABC에서 AB 위의 점 E와 AC 위의 점 D에 대하여
∠ABC = ∠ADE이고, AB = 14cm, AE = 6cm, AD = 8cm,
DC = x cm일 때, x의 값은? [4점]

선지: ① 1, ② 3/2, ③ 9/4, ④ 7/3, ⑤ 5/2

## 2) 해석

- E는 선분 AB 위의 점
- D는 선분 AC 위의 점
- AB = 14, AE = 6, AD = 8
- DC = x
- ∠ABC = ∠ADE

또한 A에서의 각은 공통으로 대응한다.

- ∠BAC = ∠DAE (AB와 AC 방향이 각각 AE와 AD와 같은 직선 위)

따라서

- △ABC ∼ △ADE

## 3) 비례식

유사 대응을 다음처럼 잡으면

- A ↔ A
- B ↔ D
- C ↔ E

그래서

AB / AD = AC / AE

값 대입:

14 / 8 = AC / 6

AC = 6 × 14 / 8 = 21/2

문제에서 AD = 8 이므로

DC = AC - AD = 21/2 - 8 = 5/2

## 4) 정답

x = 5/2 (⑤)
