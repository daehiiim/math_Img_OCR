# TECH_STACK

- **Language**: TypeScript, Python
- **Framework**: Next.js (App Router), FastAPI
- **Database**: Prisma (PostgreSQL), Supabase
- **UI**: Tailwind CSS, Shadcn UI
- **Deployment**: Vercel, google cloud run

- The above stack is the default baseline.
- Do not follow the baseline stack rigidly when a better alternative exists.
- Prefer practical, production-ready choices over fashionable ones.
- Optimize for fast execution, robustness, low operational complexity, and long-term maintainability.
